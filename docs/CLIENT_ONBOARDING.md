## 1. 제품 소개

### 한 줄 요약

**FitLink**는 헬스 트레이너의 일정 관리, 회원 관리, PT 예약, 소통을 하나의 앱으로 통합하는 모바일 웹 서비스입니다.

### 해결하려는 문제

| 기존 방식 | FitLink 방식 |
|-----------|-------------|
| 카카오톡으로 예약 조율 (분산된 소통) | 앱 내 예약 시스템으로 요청→승인→확정 자동화 |
| 수기 장부로 PT 횟수·수납 관리 | 디지털 기록으로 실시간 잔여 횟수·수납 내역 확인 |
| 구두 전달로 운동 프로그램 안내 | 운동 매뉴얼 업로드 + 오늘의 운동 배정 기능 |
| 회원별 메모를 개인 노트에 관리 | 회원 프로필에 메모 기록, 이력 축적 |

### 핵심 가치

- **효율적인 일정 관리** — 근무시간 설정, 예약 승인/거절, 캘린더 통합 조회
- **체계적인 회원 관리** — 회원 목록, 프로필 조회, 운동 메모, PT 횟수 추적
- **편리한 연결 시스템** — 초대 코드 또는 검색을 통한 트레이너-회원 매칭
- **운동 가이드 공유** — 사진·영상 포함 매뉴얼 등록, 일일 운동 배정

### 플랫폼

- **모바일 퍼스트 웹앱** — 최대 너비 480px, 스마트폰 화면에 최적화
- **PWA 지원** — 홈 화면에 추가하여 앱처럼 사용 가능
- **Flutter WebView 앱** — 하이브리드 앱으로 앱스토어 배포 예정

## 2. 사용자 유형과 역할

FitLink는 **트레이너**와 **회원** 두 가지 역할로 구분됩니다.
최초 가입 시 역할을 선택하며, 한 번 정한 역할은 변경할 수 없습니다.

### 트레이너

> 다수의 회원 일정을 관리하고 운동 가이드를 제공하는 전문가

| 할 수 있는 일 | 설명 |
|--------------|------|
| 회원 목록 관리 | 연결된 회원 전체 조회, 프로필 상세 확인 |
| 예약 승인/거절 | 회원의 PT 요청을 검토 후 확정 또는 거부 |
| 근무시간 설정 | 요일별 가능 시간·PT 단위 시간 지정 |
| 운동 매뉴얼 업로드 | 사진/영상 포함 운동법 등록·관리 |
| 오늘의 운동 배정 | 회원별 일일 운동 프로그램 텍스트 전달 |
| 운동 메모 작성 | 회원 컨디션·특이사항 기록 |
| PT 횟수 관리 | 회원 구매 횟수 등록·차감 |
| 수납 기록 | 결제 내역 수동 입력·관리 |
| 초대 코드 관리 | 회원 연결용 고유 코드 생성·공유 |
| 1:1 채팅 | 연결된 회원과 메시지·파일 송수신 |

### 회원

> 트레이너와 일정을 조율하고 운동 기록 및 매뉴얼을 확인하는 사용자

| 할 수 있는 일 | 설명 |
|--------------|------|
| 트레이너 연결 | 검색 또는 초대 코드로 트레이너와 매칭 |
| PT 예약 요청 | 트레이너 가능 시간 조회 후 예약 신청 |
| 캘린더 확인 | 본인의 예약 현황을 달력으로 조회 |
| 운동 매뉴얼 열람 | 트레이너가 업로드한 운동법 확인 |
| 오늘의 운동 확인 | 배정된 일일 운동 프로그램 조회 |
| 잔여 횟수 확인 | 남은 PT 횟수·수납 이력 조회 (읽기 전용) |
| 1:1 채팅 | 연결된 트레이너와 메시지·파일 송수신 |

### 역할별 접근 범위

| 역할 | 접근 가능 영역 | 하단 네비게이션 탭 |
|------|---------------|-------------------|
| 트레이너 | `/trainer/*` 전체 + 초대 코드 관리 | 홈 · 스케줄 · 회원 · 채팅 · 설정 |
| 회원 | `/member/*` + `/home` + 초대 코드 입력 | 홈 · 스케줄 · 채팅 · 매뉴얼 · 설정 |
| 미인증 | 로그인 화면만 접근 가능 | 없음 |

## 3. 앱 사용 흐름 — 트레이너

### 3-1. 가입부터 홈 화면까지

```
앱 실행
  → 로그인 화면 (카카오로 시작하기 버튼)
  → 카카오 OAuth 인증
  → 역할 선택 화면 ("트레이너" 선택)
  → 트레이너 프로필 입력
     · 프로필 사진 업로드
     · 이름 입력 (필수)
     · 전문 분야 선택 (재활, 근력, 다이어트, 스포츠 퍼포먼스)
  → [완료] 클릭
  → 트레이너 홈 대시보드 진입
```

### 3-2. 회원 연결 (초대 코드 방식)

```
설정 탭 → 초대 코드 관리
  → 나의 고유 초대 코드 확인
  → [코드 복사] 또는 [링크 공유] 클릭
  → 카카오톡 등으로 회원에게 코드 전달
  → 회원이 코드 입력 시 즉시 연결 완료
```

### 3-3. 근무시간 설정

```
설정 탭 → 근무시간 설정
  → 활성화할 요일 토글 (월~일)
  → 요일별 시작/종료 시간 입력
  → PT 시간 단위 선택 (30분 / 1시간 / 1시간30분 / 2시간)
  → [저장]
```

> 근무시간을 설정해야 회원이 예약 요청을 보낼 수 있습니다.

### 3-4. 예약 관리

```
예약 요청 수신 (알림)
  → 예약 관리 화면 진입
  → 대기중 예약 목록 확인
  → 개별 예약 클릭 → 회원 정보·시간 확인
  → [승인] 또는 [거절] 클릭
  → 승인된 예약은 캘린더에 자동 반영
```

### 3-5. 일상 업무 사이클

```
[매일 반복]
홈 대시보드 → 오늘의 일정 확인
  → 캘린더 탭 → 날짜별 전체 예약 조회
  → 회원 탭 → 특정 회원 선택 → 메모 작성 / 오늘의 운동 배정
  → 채팅 탭 → 회원과 소통
  → 설정 탭 → 매뉴얼 등록 / 근무시간 수정
```

## 4. 앱 사용 흐름 — 회원

### 4-1. 가입부터 홈 화면까지

```
앱 실행
  → 로그인 화면 (카카오로 시작하기 버튼)
  → 카카오 OAuth 인증
  → 역할 선택 화면 ("회원" 선택)
  → 회원 프로필 입력
     · 프로필 사진 업로드
     · 이름 입력 (필수)
     · 연락처 입력
     · 신체 정보 입력 (키, 몸무게, 나이 — 선택사항)
     · 운동 목표 작성
     · 특이사항/부상 이력 작성
  → [완료] 클릭
  → 회원 홈 대시보드 진입
```

### 4-2. 트레이너 연결

**방법 1 — 초대 코드 (즉시 연결)**
```
홈 화면 또는 초대 코드 입력 메뉴 진입
  → 트레이너에게 받은 6자리 코드 입력
  → [코드 확인] → 트레이너 정보 확인
  → [연결 확정] → 즉시 연결 완료
```

**방법 2 — 트레이너 검색 (승인 필요)**
```
트레이너 찾기 메뉴 진입
  → 이름으로 검색
  → 트레이너 카드 확인 (사진, 이름, 전문 분야)
  → [연결 요청] 클릭
  → 트레이너가 승인하면 연결 완료
```

> 회원은 동시에 1명의 트레이너와만 연결할 수 있습니다.

### 4-3. PT 예약

```
홈 또는 스케줄 탭에서 [예약하기]
  → 캘린더에서 날짜 선택 (예약 가능일 표시)
  → 시간 슬롯 목록에서 원하는 시간 선택
     (이미 예약된 시간은 회색 비활성화)
  → [예약 요청] 클릭
  → 트레이너 승인 대기 → 승인 시 확정
```

### 4-4. 일상 사용 사이클

```
[매일 반복]
홈 대시보드 → 다음 예약 일정 확인 + 잔여 PT 횟수 확인
  → 스케줄 탭 → 내 예약 현황 캘린더로 조회
  → 매뉴얼 탭 → 트레이너가 올린 운동법 열람
  → 채팅 탭 → 트레이너와 소통
  → 설정 탭 → 내 프로필 확인
```

## 5. 화면별 상세 안내

### 5-1. 공통 화면

#### 로그인 화면
- 앱 로고 이미지와 서비스 캐치프레이즈
- **[카카오로 시작하기]** 버튼 (카카오 브랜드 컬러 적용)
- 하단 이용약관 및 개인정보처리방침 링크

#### 역할 선택 화면
- 안내 문구: "어떤 역할로 사용하시겠어요?"
- **[트레이너]** 선택 카드 — 전문가 아이콘 및 설명
- **[회원]** 선택 카드 — 일반 사용자 아이콘 및 설명
- **[다음]** 버튼

#### 프로필 입력 화면

| 항목 | 트레이너 | 회원 |
|------|---------|------|
| 프로필 사진 | 원형 업로드 영역 | 원형 업로드 영역 |
| 이름 | 필수 | 필수 |
| 연락처 | — | 입력란 |
| 전문 분야 | 다중 선택 체크박스 | — |
| 신체 정보 (키/몸무게/나이) | — | 선택사항 |
| 운동 목표 | — | 텍스트 영역 |
| 특이사항/부상 이력 | — | 텍스트 영역 |

---

### 5-2. 트레이너 전용 화면

#### 트레이너 홈 대시보드
- **상단**: 환영 문구 + 프로필 요약
- **오늘의 일정 카드**: 오늘 남은 PT 예약 리스트 (시간, 회원명), [일정 전체보기] 버튼
- **예약 요청 배지**: 새로운 예약 요청 건수 + 바로가기
- **최근 메시지 섹션**: 안읽은 채팅 미리보기

#### 캘린더 (스케줄 탭)
- **월간 캘린더**: 날짜별 예약 상태 도트 표시
  - 노란 점 = 대기중 | 초록 점 = 승인됨 | 회색 점 = 완료 | 빨간 점 = 취소됨
- **날짜 선택 시**: 하단에 해당일 예약 목록 표시
- **즉시 승인/거절**: 대기중 예약 옆 [승인], [거절] 버튼 노출

#### 회원 관리 (회원 탭)
- 연결된 회원 목록 카드 (프로필 사진, 이름, 예약 통계)
- 회원 선택 시 → 상세 프로필 화면 진입
  - 회원 기본 정보 (이름, 신체 정보, 운동 목표)
  - PT 잔여 횟수 표시 바
  - 최근 운동 메모 요약

#### 예약 관리
- **필터 탭**: 전체 · 대기중 · 승인됨 · 완료
- **예약 카드**: 회원 프로필, 이름, 일정, 상태 배지
- **상태별 액션**:
  - 대기중 → [승인] / [거절]
  - 승인됨 → [완료 처리] / [취소]

#### 근무시간 설정
- 요일 토글 스위치 (월~일)
- 요일별 시작/종료 시간 선택
- PT 시간 단위: 30분 / 1시간 / 1시간30분 / 2시간 라디오 버튼
- [저장] 버튼

#### 설정
- 내 프로필 조회
- 근무시간 설정 바로가기
- 운동 매뉴얼 관리 바로가기
- 초대 코드 관리
- [로그아웃] 버튼

---

### 5-3. 회원 전용 화면

#### 회원 홈 대시보드
- **상단**: 환영 문구 + 연결된 트레이너 이름
- **예약 현황 카드**: 가장 가까운 PT 일정 (날짜, 시간), [예약 변경/취소] 바로가기
- **PT 잔여 횟수**: 남은 횟수 강조 표시 (3회 이하일 때 색상 변경)
- **오늘의 운동**: 트레이너가 배정한 운동 요약
- **미연결 상태**: "아직 담당 트레이너가 없습니다" + [트레이너 찾기] 버튼

#### PT 예약 화면
- **월간 캘린더**: 예약 가능일 표시
- **시간 슬롯 목록**: 선택한 날짜의 가능한 시간대 나열
- 이미 예약된 시간은 회색 비활성화
- [예약 요청] 버튼

#### 스케줄 (캘린더 탭)
- 본인 예약 현황을 달력에 도트로 표시
- 날짜 선택 시 해당일 예약 상세 확인

#### 운동 매뉴얼 (매뉴얼 탭)
- 카테고리 탭: 전체 · 재활 · 근력 · 다이어트 · 스포츠 퍼포먼스
- 검색바: 운동 이름으로 검색
- 매뉴얼 카드: 썸네일, 운동명, 카테고리 태그
- 상세 화면: 영상 플레이어 + 사진 갤러리 + 운동법 텍스트

#### 설정
- 내 프로필 확인
- [로그아웃] 버튼

## 6. 디자인 시스템

### 6-1. 컬러 팔레트

## 8. 기술 구성 요약

> 이 섹션은 프로젝트의 기술적 배경을 이해하기 위한 참고 자료입니다.

### 기술 스택

| 구분 | 기술 | 역할 |
|------|------|------|
| 프론트엔드 | Vue 3 (Composition API) | 화면 렌더링, 사용자 인터랙션 |
| 빌드 도구 | Vite 5 | 개발 서버 + 프로덕션 빌드 |
| 상태 관리 | Pinia | 인증 상태, 유저 정보 관리 |
| 라우팅 | Vue Router 4 | 페이지 이동 + 접근 제어 |
| 백엔드 | Supabase | 데이터베이스 + 인증 + 파일 저장소 |
| 인증 | 카카오 OAuth (PKCE) | 소셜 로그인 |
| 스타일링 | CSS Custom Properties + BEM | 디자인 시스템 토큰 기반 스타일링 |
| 언어 | JavaScript (TypeScript 미사용) | — |

### 데이터베이스 구조

| 테이블 | 설명 |
|--------|------|
| `profiles` | 사용자 기본 정보 (이름, 역할, 전화번호, 프로필 사진) |
| `trainer_profiles` | 트레이너 추가 정보 (전문 분야, 소개글) |

## 9. 용어 정리

| 용어 | 설명 |
|------|------|
| **PT (Personal Training)** | 트레이너와 회원이 1:1로 진행하는 운동 세션 |
| **트레이너** | 회원을 관리하고 운동 매뉴얼을 제공하는 전문가 사용자 |
| **회원** | 트레이너에게 운동 지도를 받는 사용자 (1명의 트레이너와만 연결) |
| **초대 코드** | 트레이너가 회원 연결을 위해 발급하는 6자리 고유 코드 |
| **슬롯** | 예약 가능한 시간 단위 (30분/1시간/1시간30분/2시간) |
| **승인/거절** | 회원의 예약 요청에 대한 트레이너의 수락/거부 |
| **매뉴얼** | 운동 자세와 팁을 담은 트레이너 제작 콘텐츠 |
| **오늘의 운동** | 트레이너가 회원에게 배정하는 일일 운동 프로그램 텍스트 |
| **수납** | PT 비용을 수동으로 기록하는 장부 기능 (실제 결제 아님) |
| **RLS** | Row Level Security — 데이터베이스 행 단위 접근 제어 |
| **PKCE** | 보안 강화된 OAuth 인증 방식 |
| **PWA** | Progressive Web App — 웹앱을 네이티브 앱처럼 사용 가능하게 하는 기술 |
