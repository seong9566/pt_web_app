## 15. 운동 프로그램 배정 — 오늘의 운동

### 개요

트레이너가 회원에게 매일 수행할 운동 과제를 자유로운 텍스트 형태로 전달합니다. 복잡한 구조화 없이 간단한 메모 형식으로 소통하며, 회원은 이를 바탕으로 개인 운동을 수행합니다.

### 사용자 흐름

**트레이너: 운동 배정**

1. 일정 탭으로 이동합니다.
2. 달력에서 날짜를 선택 합니다(기본값은 오늘입니다).
3. 승인된 예약 건이 있다면 해당 예약 건을 선택합니다.
4. "오늘의 운동 배정" 화면에서 운동 루틴을 작성합니다.
5. [저장]을 누르면 회원에게 알림이 전송됩니다.

**회원: 운동 확인**

1. 홈 화면의 '오늘의 운동' 카드에서 내용을 바로 확인하거나 [전체 보기]를 누릅니다.
2. 일정 탭의 자신의 승인된 예약 건을 선택하여 확인합니다.
3. 설정 탭의 "PT 히스토리"에서 과거의 운동 내용을 조회합니다.

### 화면 설명

1. **오늘의 운동 배정 화면 (트레이너용)**

   - 대상 정보: 회원 이름과 프로필 이미지가 상단에 표시됩니다.
   - 날짜 피커: 운동을 수행할 날짜를 선택합니다.
   - 운동 루틴 입력란 : 운동 루틴을 작성합니다(운동 제목 , 세트 수, 횟수)

2. **배정 이력 화면 (트레이너용)**

   - 리스트 뷰: 날짜별로 작성된 운동 내용의 일부를 미리보기 형태로 나열합니다.
   - 관리 버튼: 각 항목 옆에 [수정]과 [삭제] 버튼이 배치됩니다.

3. **오늘의 운동 상세 화면 (회원용)**
   - 날짜 표시: 해당 운동이 배정된 날짜가 상단에 나타납니다.
   - 내용 영역: 트레이너가 작성한 전체 텍스트를 가독성 있게 보여줍니다.
   - 이동 컨트롤: 이전 날짜나 다음 날짜의 운동으로 쉽게 이동할 수 있는 화살표 버튼을 제공합니다.

### 비즈니스 규칙

- 운동 배정 및 관리 권한은 트레이너에게만 있습니다.
- 회원은 배정된 내용을 수정하거나 삭제할 수 없습니다.
- 하루에 하나의 운동 프로그램만 배정 가능하며, 재작성 시 기존 내용을 덮어씁니다.
- 배정 완료 시 해당 회원에게 즉시 인앱 알림을 보냅니다.

### 엣지 케이스

- 중복 배정: 이미 내용이 있는 날짜에 저장 시 "기존 내용을 덮어쓰시겠습니까?"라는 확인창을 띄웁니다.
- 미래 날짜 배정: 며칠 치 운동을 미리 작성하여 배정할 수 있습니다.
- 연결 해제: 트레이너와 회원의 연결이 끊겨도 기존에 배정된 이력은 데이터베이스에 보관됩니다.

### 범위 외

- 회원이 운동 완료 버튼을 누르거나 수행 결과를 기록하는 기능은 제외합니다.
- 매뉴얼 화면에서 특정 운동을 끌어다 놓아 배정하는 드래그 앤 드롭 기능은 포함하지 않습니다.

## 16. 회원 운동 메모

### 개요

트레이너가 회원의 특이사항, 컨디션, 운동 시 주의점 등을 기록하는 전용 공간입니다. 트레이너의 관리 효율을 높이고 회원에게는 본인의 상태에 대한 피드백을 전달하는 용도로 활용됩니다.

### 사용자 흐름

**트레이너: 메모 작성 및 관리**

1. 회원 상세 정보 화면에서 '메모' 섹션을 확인합니다.
2. 우측 하단의 FAB.[메모 추가] 버튼을 눌러 날짜와 메모 내용을 입력한 뒤 저장합니다.
3. 목록에서 특정 메모를 탭하여 상세 내용을 확인하거나 수정, 삭제합니다.

### 화면 설명

1. **메모 목록 화면 (트레이너용)**

   - 회원 식별: 상단에 회원 이름이 명시됩니다.
   - 카드 리스트: 작성 날짜와 메모 내용의 앞부분(약 2줄)을 보여줍니다.
   - 추가 버튼: 우측 하단에 플로팅 형태로 [메모 추가] 버튼이 위치합니다.

2. **메모 작성/수정 창**

   - 날짜 선택: 메모와 연결된 날짜 및 시간을 지정합니다.
   - 텍스트 필드: 제약 없는 자유로운 작성이 가능한 입력 창입니다.

### 비즈니스 규칙

- 메모의 모든 편집 권한(생성, 수정, 삭제)은 트레이너에게 있습니다.
- 한 날짜에 여러 개의 메모를 남길 수 있습니다.
- 메모는 텍스트 위주로 구성됩니다.

### 엣지 케이스

- 메모 삭제: 실수 방지를 위해 삭제 전 반드시 확인 절차를 거칩니다.

### 범위 외

- 회원이 트레이너에게 메모를 남기는 기능은 포함하지 않습니다.
- 무게 변화나 체지방률 등 수치 데이터의 시계열 그래프 표시는 제외합니다.
- 특정 키워드로 메모를 검색하는 기능은 구현하지 않습니다.
