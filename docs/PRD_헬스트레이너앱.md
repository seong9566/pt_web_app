## 1. 제품 개요

- **제품명**: PT 매니저 (가칭)
- **목적**: 헬스 트레이너의 일정 관리, 회원 관리, PT 예약, 소통을 통합 지원하여 업무 효율성을 높이고 회원의 운동 경험을 개선합니다.
- **타깃 사용자**:
  - **헬스 트레이너**: 다수의 회원 일정을 관리하고 운동 가이드를 제공하는 전문가
  - **PT 회원**: 트레이너와 일정을 조율하고 운동 기록 및 매뉴얼을 확인하는 사용자
- **플랫폼**: Vue.js 기반 웹 서비스 및 Flutter WebView 앱 (하이브리드 앱)
- **핵심 가치 제안**: 번거로운 수기 관리와 카카오톡 분산 소통을 하나의 앱으로 통합하여 체계적인 PT 서비스를 제공합니다.
- **사용자 유형 정의**:
  - **트레이너**: 회원 목록 관리, 예약 승인/거절, 운동 매뉴얼 업로드, 오늘의 운동 배정, 1:1 채팅
  - **회원**: 트레이너 검색 및 연결, PT 예약 요청, 운동 매뉴얼 열람, 배정된 운동 확인, 1:1 채팅

## 2. 인증 및 온보딩

### 개요

카카오 OAuth를 활용한 간편 소셜 로그인을 제공합니다. 최초 가입 시 사용자는 트레이너와 회원 중 자신의 역할을 선택하고 필수 프로필 정보를 입력하여 서비스를 시작합니다.

### 사용자 흐름

- **최초 로그인**: 앱 실행 → [카카오로 시작하기] 버튼 클릭 → 카카오 인증 완료 → 역할 선택 화면 → 프로필 정보 입력 → 홈 화면 진입
- **재로그인**: 앱 실행 → [카카오로 시작하기] 버튼 클릭 → 카카오 인증 완료 → 홈 화면 진입 (역할 선택 및 프로필 입력 단계 생략)
- **로그아웃 후 재진입**: 로그인 화면 노출 → 카카오 인증 → 홈 화면 진입

### 화면 설명

1.  **로그인 화면**
    - **UI 요소**: 앱 로고 이미지, 서비스 캐치프레이즈 텍스트, [카카오로 시작하기] 버튼 (카카오 브랜드 컬러 적용), 하단 이용약관 및 개인정보처리방침 링크
2.  **역할 선택 화면**
    - **UI 요소**: 상단 안내 문구 "어떤 역할로 사용하시겠어요?", [트레이너] 선택 카드 (전문가 아이콘 및 설명 포함), [회원] 선택 카드 (일반 사용자 아이콘 및 설명 포함), [다음] 버튼
3.  **트레이너 프로필 입력 화면**
    - **UI 요소**: 프로필 사진 업로드 영역 (원형 가이드), 이름 입력란, 전문 분야 다중 선택 체크박스 (재활, 근력, 다이어트, 스포츠 퍼포먼스), [완료] 버튼
4.  **회원 프로필 입력 화면**
    - **UI 요소**: 프로필 사진 업로드 영역, 이름 입력란, 연락처 입력란, 신체 정보 입력란 (키 cm, 몸무게 kg, 나이), 운동 목표 텍스트 영역, 특이사항/부상 이력 텍스트 영역, [완료] 버튼

### 비즈니스 규칙

- 하나의 카카오 계정은 하나의 역할(트레이너 또는 회원)만 가질 수 있습니다.
- 선택한 역할은 가입 완료 후 변경이 불가능합니다.
- 이름 및 프로필 입력은 필수 단계이며 건너뛸 수 없습니다.
- 회원의 신체 정보(키, 몸무게, 나이)는 선택 사항으로 비워둘 수 있습니다.

### 엣지 케이스

- **카카오 인증 실패**: 카카오 서버 오류 또는 인증 취소 시 "인증에 실패했습니다" 안내 문구와 [다시 시도] 버튼 표시
- **네트워크 오류**: 가입 처리 중 네트워크 단절 시 상태 안내 및 재시도 유도
- **중복 가입 시도**: 이미 가입된 계정인 경우 역할 선택 단계를 건너뛰고 기존 데이터로 홈 화면 이동

### 범위 외

- 이메일/비밀번호 기반의 자체 회원가입 기능
- 네이버, 구글, 애플 등 타 소셜 로그인 연동
- 가입 후 사용자 역할 변경 기능
- 다중 소셜 계정 연동 및 관리

## 3. 프로필 관리

### 개요

트레이너와 회원이 자신의 정보를 관리하고, 트레이너가 관리 중인 회원의 정보를 열람할 수 있는 기능입니다.

### 사용자 흐름

- **본인 프로필 수정**: 설정 메뉴 진입 → 내 프로필 메뉴 선택 → [수정] 버튼 클릭 → 정보 변경 → [저장] 클릭
- **트레이너의 회원 정보 조회**: 회원 관리 목록 진입 → 특정 회원 카드 선택 → 회원 프로필 상세 정보 확인

### 화면 설명

1.  **트레이너 프로필 조회 화면**
    - **UI 요소**: 원형 프로필 이미지, 이름 텍스트, 전문 분야 태그 리스트, 현재 연결된 회원 수 카운트, [프로필 수정] 버튼
2.  **트레이너 프로필 수정 화면**
    - **UI 요소**: 이미지 변경 버튼, 이름 입력란, 전문 분야 체크박스 리스트, [저장] 버튼, [취소] 버튼
3.  **회원 프로필 조회 화면**
    - **UI 요소**: 프로필 이미지, 이름, 연락처, 신체 데이터 요약(키/몸무게/나이), 운동 목표 및 특이사항 텍스트, [프로필 수정] 버튼 (본인 열람 시에만 노출)
4.  **회원 프로필 수정 화면**
    - **UI 요소**: 이미지 업로드 버튼, 이름/연락처 입력란, 키/몸무게/나이 숫자 입력란, 운동 목표/특이사항 멀티라인 입력란, [저장] 및 [취소] 버튼
5.  **트레이너 전용 회원 프로필 조회**
    - **UI 요소**: 회원 기본 정보, PT 잔여 횟수 표시 바, 최근 트레이너가 작성한 운동 메모 요약 섹션

### 비즈니스 규칙

- 프로필 수정 권한은 본인에게만 있습니다.
- 트레이너는 자신과 연결된 회원에 한해서만 프로필 상세 정보를 조회할 수 있습니다.
- 이름은 시스템 식별 및 소통을 위해 반드시 입력해야 합니다.

### 엣지 케이스

- **이미지 미등록**: 사용자가 프로필 사진을 등록하지 않은 경우 시스템 기본 아바타 이미지를 표시합니다.
- **필수값 누락**: 수정 시 이름을 비워둘 경우 "이름을 입력해주세요"라는 유효성 검사 메시지를 출력하고 저장을 차단합니다.

### 범위 외

- 자격증 증명 파일 업로드 및 검증 프로세스
- 상세 경력 사항 및 포트폴리오 섹션
- 외부 SNS(인스타그램 등) 링크 연동
- 회원 간의 프로필 조회 기능

## 4. 인앱 알림

### 개요

서비스 내 주요 이벤트 발생 시 사용자에게 정보를 전달하는 시스템입니다. 별도의 OS 푸시 알림 없이 앱 내부의 알림 센터를 통해 작동합니다.

### 사용자 흐름

- **알림 확인**: 새로운 알림 발생 → 하단 네비게이션 바 '알림' 아이콘에 빨간색 숫자 배지 생성 → 알림 탭 클릭 → 알림 리스트 확인 → 개별 알림 클릭 → 관련 기능 화면으로 이동

### 화면 설명

1.  **네비게이션 바 배지**
    - **UI 요소**: 하단 탭 바의 알림 아이콘 우측 상단에 표시되는 빨간색 원형 숫자 배지
2.  **알림 센터 화면**
    - **UI 요소**: "알림" 상단 타이틀, [전체 읽음 처리] 버튼, 알림 목록 (최신순 정렬), 알림 항목 카드(이벤트 아이콘, 알림 내용 텍스트, 발생 시간, 읽음/안읽음 상태 표시 색상)
3.  **빈 알림 화면**
    - **UI 요소**: "새로운 알림이 없습니다" 안내 텍스트 및 일러스트

### 비즈니스 규칙

- 알림은 앱이 실행 중인 상태에서 센터를 통해 확인 가능합니다 (OS 레벨 푸시 없음).
- 알림 항목을 클릭하면 해당 이벤트와 연관된 상세 화면(예: 예약 상세, 채팅방 등)으로 자동 이동합니다.
- 알림 내역은 발생일로부터 30일간 보관 후 자동 삭제됩니다.

### 엣지 케이스

- **데이터 삭제 후 알림 클릭**: 알림의 대상이 되는 데이터(예: 취소된 예약)가 이미 삭제된 경우 "해당 정보를 찾을 수 없습니다"라는 팝업을 노출합니다.
- **대량 알림 발생**: 단시간에 많은 알림이 발생할 경우 최신순으로 리스트를 갱신합니다.

### 범위 외

- 스마트폰 상단 바에 표시되는 OS 푸시 알림 서비스
- 알림 유형별 온/오프 설정 기능
- 개별 알림 삭제 기능
- 이메일 또는 문자 메시지(SMS) 연동 알림

## 5. 트레이너 검색 및 초대

### 개요

회원이 함께 운동할 트레이너를 찾고 연결을 요청하는 기능입니다. 이름 검색이나 트레이너가 발급한 초대 코드를 통해 연결이 이루어집니다.

### 사용자 흐름

- **검색 연결**: 트레이너 찾기 메뉴 → 이름 입력 또는 필터 선택 → 트레이너 카드 확인 → [연결 요청] 클릭 → 트레이너가 알림 확인 후 승인
- **초대 코드 연결**: 트레이너로부터 코드 수신 → 앱 내 초대 코드 입력 창에 입력 → 트레이너 정보 확인 → [연결하기] 클릭 → 즉시 연결 완료

### 화면 설명

1.  **트레이너 검색 화면**
    - **UI 요소**: 상단 검색바, 트레이너 리스트 카드 (사진, 이름, 분야 태그, [연결 요청] 버튼)
2.  **트레이너 상세 프로필 팝업**
    - **UI 요소**: 상세 정보 텍스트, [연결 요청] 버튼, 이미 연결된 경우 [연결됨] 상태 표시
3.  **초대 코드 입력 화면**
    - **UI 요소**: 6자리 영문/숫자 혼합 코드 입력란, [코드 확인] 버튼, 확인 후 나타나는 트레이너 요약 카드, [연결 확정] 버튼
4.  **트레이너용 초대 코드 관리**
    - **UI 요소**: 나의 고유 초대 코드 텍스트, [코드 복사] 버튼, 카카오톡 등으로 전송하기 위한 [링크 공유] 버튼

### 비즈니스 규칙

- 회원은 동시에 단 한 명의 트레이너와만 연결 상태를 유지할 수 있습니다.
- 검색을 통한 연결은 트레이너의 '승인' 단계가 필수입니다.
- 초대 코드를 통한 연결은 트레이너가 이미 의사를 밝힌 것으로 간주하여 즉시 연결됩니다.
- 트레이너는 자신의 초대 코드를 언제든 재발급하여 이전 코드를 무효화할 수 있습니다.

### 엣지 케이스

- **잘못된 코드 입력**: 존재하지 않거나 만료된 코드를 입력한 경우 "유효하지 않은 코드입니다"라는 에러 메시지를 표시합니다.
- **중복 연결 시도**: 이미 다른 트레이너와 연결된 회원이 요청을 보낼 경우 "기존 연결을 해제한 후 시도해주세요"라고 안내합니다.

### 범위 외

- 트레이너의 위치 기반 지도 검색 기능
- 트레이너의 별점 순, 후기 순 정렬 기능
- 유료 광고를 통한 트레이너 상단 노출

## 6. 계정 관리

### 개요

사용자의 서비스 이용 종료 및 트레이너-회원 간의 관계 해제를 관리하는 기능입니다.

### 사용자 흐름

- **로그아웃**: 설정 화면 → [로그아웃] 클릭 → 확인 팝업 → 로그인 초기 화면 이동
- **계정 삭제(탈퇴)**: 설정 → 계정 관리 → [계정 삭제] 클릭 → 유의사항 확인 및 동의 → [최종 탈퇴] 클릭
- **연결 해제**: 회원 목록 또는 설정 메뉴 → [연결 해제] 버튼 클릭 → 사유 선택(선택사항) → 해제 완료

### 화면 설명

1.  **설정 메뉴 화면**
    - **UI 요소**: 개인정보 설정, 알림 설정 리스트, [로그아웃] 링크, [계정 관리] 메뉴
2.  **계정 관리 화면**
    - **UI 요소**: [계정 삭제] 버튼 (붉은색 강조)
3.  **계정 삭제 확인 팝업**
    - **UI 요소**: 탈퇴 시 삭제되는 데이터 안내 문구, "정말 탈퇴하시겠습니까?" 질문, [취소] 버튼, [탈퇴하기] 버튼
4.  **연결 해제 확인 화면**
    - **UI 요소**: 해제 대상 이름 정보, 해제 후 데이터 보관 안내 문구, [해제 확정] 버튼, [유지하기] 버튼

### 비즈니스 규칙

- 계정 삭제 시 개인식별이 가능한 모든 프로필 정보는 즉시 파기됩니다.
- 트레이너와 회원의 연결이 해제되더라도, 기존의 채팅 내역이나 예약 기록은 아카이브 형태로 시스템에 남습니다 (단, 서로에게는 더 이상 노출되지 않습니다).
- 탈퇴한 사용자의 데이터는 복구가 불가능합니다.

### 엣지 케이스

- **진행 중인 예약 존재 시**: 승인된 예약이 남아있는 상태에서 연결을 해제할 경우 해당 예약들이 자동으로 취소됨을 사전에 경고합니다.
- **재가입**: 탈퇴 후 동일한 카카오 계정으로 재가입할 수 있으나, 기존 데이터와는 연결되지 않는 신규 계정으로 생성됩니다.

### 범위 외

- 30일간 유예 기간을 두는 계정 삭제 정책 (즉시 삭제 원칙)
- 사용자 데이터 전체 백업 및 내보내기 기능
- 계정 일시 정지(휴면) 기능

## 7. 대시보드 — 홈 화면

### 개요

앱 접속 시 가장 먼저 노출되는 화면으로, 사용자의 역할에 맞춰 현재 가장 중요한 정보를 요약하여 보여줍니다.

### 사용자 흐름

- **진입**: 로그인/앱 실행 → 홈 화면 자동 노출 → 각 요약 카드 클릭 → 상세 기능(캘린더, 채팅, 매뉴얼 등)으로 이동

### 화면 설명

1.  **트레이너 홈 화면**
    - **UI 요소**:
      - **상단**: 트레이너 환영 문구 및 프로필 요약
      - **오늘의 일정 카드**: 오늘 남은 PT 예약 리스트 (시간, 회원명 표시), [일정 전체보기] 버튼
      - **예약 요청 배지**: 새로운 예약 요청 건수 표시 및 목록 바로가기
      - **최근 메시지 섹션**: 안읽은 채팅 내역 미리보기
2.  **회원 홈 화면**
    - **UI 요소**:
      - **상단**: 회원 환영 문구 및 연결된 트레이너 이름
      - **예약 현황 카드**: 가장 가까운 PT 일정(날짜, 시간, 장소 정보), [예약 변경/취소] 바로가기
      - **PT 잔여 횟수**: 현재 남은 횟수를 큰 숫자로 강조하여 표시
      - **오늘의 운동**: 트레이너가 작성해준 오늘 수행할 운동 요약 텍스트
3.  **미연결 회원용 홈**
    - **UI 요소**: "아직 담당 트레이너가 없습니다" 문구, [트레이너 찾기] 버튼, 서비스 이용 가이드 카드

### 비즈니스 규칙

- 홈 화면의 데이터는 앱 실행 시마다 실시간으로 갱신됩니다.
- 가장 시급한 정보(예: 곧 시작할 예약)를 상단에 배치합니다.
- 잔여 PT 횟수가 3회 이하로 남은 경우 시각적으로 강조(색상 변경 등)하여 재등록을 유도합니다.

### 엣지 케이스

- **일정 없음**: 오늘 예정된 PT가 없는 경우 "오늘은 일정이 없습니다. 휴식을 즐기세요!"와 같은 친근한 문구를 노출합니다.
- **장기 미접속**: 오랜 기간 활동이 없는 경우 새로운 운동 매뉴얼이나 추천 콘텐츠를 추천하는 카드를 노출할 수 있습니다.

### 범위 외

- 상세한 신체 변화 그래프(체지방률, 근육량 등) 표시
- 트레이너의 일일 수익 통계 화면
- 회원들 간의 랭킹이나 커뮤니티 피드 리스트

## 8. 트레이너 근무시간 및 예약 단위 설정

### 개요

트레이너가 회원들의 예약이 가능한 요일과 시간대를 미리 설정하는 기능입니다. 트레이너는 PT 1회당 소요되는 시간 단위도 지정할 수 있으며, 회원은 오직 이 범위 안에서만 예약 요청을 보낼 수 있습니다.

### 사용자 흐름

- **근무시간 설정**: 트레이너가 설정 메뉴에 진입합니다. 근무시간 설정을 선택하고 활성화할 요일을 정합니다. 각 요일별로 시작 시간과 종료 시간을 입력하고 PT 시간 단위를 고른 뒤 저장합니다.
- **휴무일 설정**: 트레이너가 캘린더 화면에서 특정 날짜를 선택합니다. 휴무 설정 버튼을 눌러 해당 날짜의 예약을 일시적으로 차단합니다.

### 화면 설명

1.  **근무시간 설정 화면**
    - 타이틀: "근무시간 설정"
    - 요일 선택: 월, 화, 수, 목, 금, 토, 일 토글 스위치
    - 시간 입력: 요일별 시작/종료 시간 선택 드롭다운 (예: 09:00 ~ 18:00)
    - 단위 선택: 30분, 1시간, 1시간 30분, 2시간 라디오 버튼
    - 버튼: [저장]
2.  **휴무일 설정 화면**
    - 뷰: 월간 캘린더
    - 동작: 날짜 선택 시 하단에 [휴무 설정] 또는 [휴무 해제] 토글 표시
    - 표시: 휴무로 지정된 날짜는 캘린더에서 별도 색상으로 강조
    - 버튼: [변경 사항 저장]

### 비즈니스 규칙

- 근무시간을 하나도 설정하지 않으면 회원이 예약을 요청할 수 없습니다.
- 예약은 트레이너가 정한 시간 단위로만 가능합니다. 1시간 단위로 설정했다면 회원은 30분 예약을 할 수 없습니다.
- 이미 승인된 예약이 있는 상태에서 근무시간을 변경해도 기존 예약은 취소되지 않습니다. 변경 이후의 새 예약 신청에만 적용됩니다.
- 기본적으로 주간 단위로 반복 설정되나, 휴무 설정을 통해 특정 날짜만 예외 처리가 가능합니다.

### 엣지 케이스

- **중복 예약 충돌**: 근무시간 변경 시 기존 예약과 겹치는 경우 트레이너에게 경고 알림을 띄우고 기존 예약은 유지합니다.
- **시간 설정 오류**: 종료 시간이 시작 시간보다 빠르거나 같게 설정하려고 하면 "유효하지 않은 시간 범위입니다"라는 안내를 표시합니다.

### 범위 외

- 매주 고정된 시간에 반복되는 정기 PT 자동 등록 기능
- 예약 취소 발생 시 대기자에게 알리는 명단 기능
- 동일 시간에 여러 명을 받는 오버부킹 설정

## 9. PT 예약 시스템

### 개요

회원이 트레이너의 스케줄을 보고 PT를 신청하며, 트레이너가 이를 승인하거나 거절하여 일정을 확정하는 시스템입니다. 예약 상태는 대기중, 승인됨, 거절됨, 취소됨, 완료의 5가지로 관리됩니다.

### 사용자 흐름

- **예약 요청(회원)**: 홈이나 캘린더에서 예약하기 버튼을 누릅니다. 트레이너의 가능 날짜와 시간 슬롯을 확인하고 원하는 시간을 골라 요청을 보냅니다.
- **승인 및 거절(트레이너)**: 예약 요청 알림을 확인합니다. 상세 내역을 보고 승인 버튼을 누르면 확정되고, 거절 버튼을 누르면 사유를 적고 취소합니다.
- **취소**: 예약 당사자(트레이너 또는 회원)가 예약 상세 페이지에서 취소 버튼을 눌러 일정을 취소합니다.

### 화면 설명

1.  **예약 가능 시간 조회 화면(회원용)**
    - 상단: 월간 캘린더 (예약 가능일 표시)
    - 하단: 선택한 날짜의 시간 슬롯 목록 (예: 10:00, 11:00...)
    - 상태 표시: 이미 예약된 칸은 회색으로 비활성화
    - 버튼: [예약 요청]
2.  **예약 요청 확인 화면(회원용)**
    - 정보: 트레이너 성함, 날짜와 시간, PT 시간 단위 요약
    - 버튼: [확인], [돌아가기]
3.  **예약 목록 화면(트레이너용)**
    - 필터: 전체, 대기중, 승인됨, 완료 탭
    - 리스트: 회원 프로필, 이름, 일정, 상태 배지
    - 정렬: 대기중인 요청을 가장 상단에 배치
4.  **예약 상세 화면(트레이너용)**
    - 정보: 회원 사진, 이름, 잔여 PT 횟수
    - 상태: 현재 예약 상태 배지
    - 버튼: 대기중일 때 [승인]/[거절], 승인됨일 때 [취소]
5.  **예약 내역 화면(회원용)**
    - 리스트: 날짜/시간, 트레이너 정보, 상태 배지
    - 버튼: 승인됨 상태의 예약에 대해 [취소] 버튼 노출
6.  **거절 사유 입력 팝업**
    - 입력: "거절 사유를 입력해주세요 (선택)" 필드
    - 버튼: [거절 완료], [닫기]

### 비즈니스 규칙

- 상태 흐름은 대기중에서 승인됨 또는 거절됨으로 진행됩니다. 승인됨 이후에는 완료나 취소됨으로 바뀔 수 있습니다.
- 취소는 PT 시작 전이라면 별도의 시간 제한 없이 언제든 가능합니다.
- 예약된 시간이 지나면 시스템이 자동으로 상태를 완료로 변경합니다.
- 거절 사유는 반드시 입력할 필요는 없습니다.
- 잔여 PT 횟수가 없어도 예약 요청은 가능하나, 회원에게 잔여 횟수가 부족하다는 경고를 줍니다.

### 엣지 케이스

- **동시 예약 시도**: 두 회원이 같은 시간을 거의 동시에 요청할 경우, 먼저 서버에 도달한 요청만 대기중으로 등록하고 나머지는 실패 처리합니다.
- **휴무일 설정 시 기존 예약**: 예약이 이미 잡힌 날을 나중에 휴무로 바꿔도 기존 예약은 자동으로 취소되지 않습니다. 트레이너가 직접 확인 후 취소해야 합니다.

### 범위 외

- 매주 같은 시간대를 자동으로 잡는 기능
- 예약 취소 시 빈자리를 채우는 대기 명단
- 기존 예약을 직접 다른 시간으로 옮기는 변경 기능 (취소 후 새로 예약해야 함)

## 10. 캘린더

### 개요

전체적인 PT 일정을 달력 형태로 한눈에 파악하는 기능입니다. 트레이너는 관리하는 모든 회원의 일정을 보고, 회원은 본인의 일정만 확인합니다.

### 사용자 흐름

- 하단 네비게이션에서 캘린더 탭을 누르면 이번 달 일정이 나옵니다.
- 특정 날짜를 누르면 하단에 해당 날의 예약 목록이 나열됩니다.
- 목록에서 항목을 선택하면 예약 상세 화면으로 연결됩니다.
- 트레이너는 달력 하단 목록에서 대기중인 예약을 즉시 승인하거나 거절할 수 있습니다.

### 화면 설명

1.  **캘린더 메인 화면**
    - 상단: 월 이동 화살표, 현재 연월 표시
    - 전환: 월간, 주간, 일간 뷰 선택 탭
    - 월간 뷰: 날짜 칸 안에 예약 상태별 색상 점(도트) 표시
    - 상태 색상: 대기중(노랑), 승인됨(초록), 완료(회색), 취소됨(빨강)
2.  **주간 및 일간 뷰**
    - 형태: 시간대별 타임라인 방식
    - 표시: 예약 블록 안에 회원 이름과 시간 기재
3.  **일별 예약 목록 패널**
    - 리스트: 시간, 상대방 이름, 상태 배지
    - 기능: 트레이너에게는 대기중인 항목 옆에 [승인], [거절] 버튼을 바로 노출

### 비즈니스 규칙

- 트레이너는 본인에게 연결된 모든 회원의 예약을 통합하여 봅니다.
- 회원은 자신이 예약한 내역만 달력에 표시됩니다.
- 별도의 리마인더 알림은 보내지 않으며 인앱 알림 함에서 확인해야 합니다.
- 지나간 날짜의 일정도 그대로 보관되어 완료나 취소 이력을 확인할 수 있습니다.

### 엣지 케이스

- **예약 없는 날**: 하단 패널에 "등록된 예약이 없습니다"라는 메시지를 띄웁니다.
- **휴무일 표시**: 트레이너가 지정한 휴무일은 달력 배경색을 다르게 하여 예약 불가능함을 알립니다.

### 범위 외

- 외부 캘린더(구글, 애플 등)와의 연동
- 개인적인 할 일(To-do) 입력 기능
- 일정 리마인더 알림 설정

## 11. 1:1 채팅

### 개요

트레이너와 회원이 메시지를 주고받는 소통 창구입니다. 텍스트뿐만 아니라 운동 영상이나 식단 사진 등의 파일 전송을 지원합니다.

### 사용자 흐름

- 채팅 탭에서 대화 목록을 확인하고 상대를 선택해 입장합니다.
- 메시지를 입력하고 전송 버튼을 누릅니다.
- 하단의 [+] 버튼을 눌러 갤러리나 파일 보관함에 있는 자료를 선택해 보냅니다.

### 화면 설명

1.  **채팅 목록 화면**
    - 리스트: 프로필 사진, 이름, 최근 메시지, 전송 시간, 읽지 않은 메시지 수 숫자 배지
    - 강조: 새로운 메시지가 있는 채팅방은 폰트를 두껍게 표시
2.  **채팅방 화면**
    - 상단: 상대방 이름과 프로필 사진, 나가기 버튼
    - 대화창: 내 메시지는 오른쪽(파랑), 상대 메시지는 왼쪽(회색) 배치
    - 첨부: 이미지 썸네일 표시, 파일은 아이콘과 파일명 표시
    - 입력창: 첨부 버튼, 텍스트 입력 필드, 전송 아이콘
3.  **파일 첨부 옵션 팝업**
    - 메뉴: [사진/동영상 선택], [파일 선택]
4.  **이미지 뷰어**
    - 기능: 사진을 전체 화면으로 확대, [기기에 저장] 버튼 제공

### 비즈니스 규칙

- 트레이너와 회원 간의 1:1 대화만 가능하며 그룹 채팅은 지원하지 않습니다.
- 이미지 파일은 개당 최대 10MB까지 보낼 수 있습니다.
- 일반 파일(문서, 동영상 등)은 개당 최대 50MB로 제한합니다.
- 지원 형식은 이미지(jpg, png, gif, webp), 문서(pdf, docx, xlsx), 영상(mp4)입니다.
- 연결이 해제된 상대와는 새로운 메시지를 주고받을 수 없으나 이전 기록은 조회 가능합니다.
- 상대방이 메시지를 읽었는지 확인하는 숫자 1 표시

### 엣지 케이스

- **용량 초과**: 제한된 크기보다 큰 파일을 올리면 "파일 용량이 너무 큽니다"라고 안내합니다.
- **지원하지 않는 형식**: 허용되지 않은 확장자 파일을 올리면 전송을 차단하고 사유를 알립니다.
- **전송 실패**: 인터넷 연결 문제로 실패하면 메시지 옆에 재시도 아이콘을 띄웁니다.

### 범위 외

- 실시간 음성 메시지나 영상 통화
- 메시지 삭제나 수정 기능
- 이모지로 반응하는 리액션 기능

## 12. PT 횟수 관리

### 개요

회원이 구매한 PT 횟수를 등록하고 남은 횟수를 추적하는 기능입니다. 트레이너가 수동으로 관리하며 회원은 이를 조회하여 본인의 잔여 횟수를 파악합니다.

### 사용자 흐름

- **트레이너**: 회원 목록에서 특정 회원을 선택해 횟수 관리 탭으로 갑니다. 횟수 추가 버튼을 눌러 구매 수량을 넣거나, 차감 버튼을 눌러 사용량을 입력합니다.
- **회원**: 홈 화면이나 본인 프로필 영역에서 현재 남은 PT 횟수를 확인합니다.

### 화면 설명

1.  **횟수 관리 화면(트레이너용)**
    - 요약: 총 등록 횟수와 잔여 횟수를 큰 글씨로 강조
    - 버튼: [+ 횟수 추가], [- 횟수 차감]
    - 이력: 날짜별 횟수 변동 내역 리스트 (예: 2026-03-01 10회 추가)
2.  **입력 팝업**
    - 입력: 숫자 입력 필드, 간단한 메모 필드(예: "재등록")
    - 버튼: [적용], [취소]
3.  **잔여 횟수 확인 화면(회원용)**
    - 표시: "남은 PT 횟수는 00회입니다" 문구
    - 상세: 등록한 전체 횟수와 지금까지 사용한 횟수 안내

### 비즈니스 규칙

- 횟수의 직접적인 가감 권한은 트레이너에게만 있습니다.
- 회원은 본인의 남은 횟수와 변경 이력을 읽기 전용으로만 봅니다.
- 남은 횟수가 0회가 되어도 예약을 아예 막지는 않지만, 예약 요청 시 "횟수가 부족합니다"라는 경고창을 보여줍니다.
- 잔여 횟수가 마이너스가 되지 않도록 현재 남은 수보다 더 많은 양을 차감할 수 없습니다.
- 수업이 끝날 때마다 자동으로 1회씩 차감되는 자동 시스템

### 엣지 케이스

- **과다 차감**: 남은 횟수가 3회인데 5회를 빼려고 하면 입력을 막고 오류 메시지를 보여줍니다.
- **이력 누락**: 실수로 잘못 입력한 경우 삭제 기능은 제공하지 않으며, 반대되는 수치를 다시 입력하여 보정해야 합니다.

### 범위 외

- PT 이용권의 유효 기간이나 만료 알림
- 환불에 따른 횟수 정산 기능

## 13. 수납 기록

### 개요

트레이너가 회원에게 받은 비용을 장부처럼 기록하는 수동 관리 도구입니다. 앱 내에서 실제로 돈이 오가는 결제 시스템은 포함되지 않습니다.

### 사용자 흐름

- 트레이너가 회원 관리 메뉴에서 수납 기록 탭을 선택합니다.
- 수납 추가 버튼을 눌러 금액, 날짜, 어떤 명목인지 적어 저장합니다.
- 회원은 자신의 메뉴에서 지금까지 낸 수납 내역을 리스트로 확인합니다.

### 화면 설명

1.  **수납 기록 목록(트레이너용)**
    - 요약: 해당 회원의 누적 수납 합계 금액
    - 리스트: 결제 날짜, 금액, 메모 요약
    - 버튼: 플로팅 버튼 형태의 [+] 추가 버튼
2.  **수납 등록 화면**
    - 입력: 금액(숫자), 날짜 선택 달력, 메모 칸(예: "10회분 결제")
    - 버튼: [저장], [취소]
3.  **수납 내역 조회(회원용)**
    - 리스트: 날짜와 금액, 메모를 최신순으로 정렬하여 표시

### 비즈니스 규칙

- 수납 정보를 넣거나 지우는 권한은 트레이너만 가집니다.
- 회원은 트레이너가 입력해준 내역을 조회만 할 수 있습니다.
- 금액과 날짜는 필수 항목이며 메모는 비워둘 수 있습니다.
- 잘못 적은 기록은 삭제한 뒤 다시 등록해야 합니다.

### 엣지 케이스

- **내역 부재**: 수납 기록이 하나도 없다면 "기록된 수납 내역이 없습니다"라고 화면을 채웁니다.
- **0원 입력**: 금액에 0이나 음수를 적으려 하면 저장되지 않도록 유효성 검사를 합니다.

### 범위 외

- 신용카드나 계좌이체 등 실제 돈을 주고받는 결제 기능
- 영수증 파일 생성이나 외부 전송
- 세금 계산을 위한 자동 합산 시스템
- 환불 금액의 자동 계산 처리

## 14. 운동 매뉴얼

### 개요

트레이너가 운동 영상, 사진, 설명을 업로드하여 자신만의 운동 매뉴얼을 구축합니다. 등록된 매뉴얼은 트레이너와 회원 모두가 자유롭게 열람할 수 있으며, 카테고리 분류와 검색 기능을 통해 필요한 운동 정보를 쉽게 찾을 수 있습니다.

### 사용자 흐름

**트레이너: 매뉴얼 업로드**

1. 매뉴얼 탭에서 [+ 매뉴얼 추가] 버튼을 누릅니다.
2. 카테고리(재활, 근력, 다이어트, 스포츠 퍼포먼스)를 선택합니다.
3. 운동 이름을 입력합니다.
4. 자세 설명, 추천 세트/횟수, 주의사항 등 상세 내용을 작성합니다.
5. 사진 파일을 업로드합니다(여러 장 가능).
6. 영상 추가 방식을 선택합니다(직접 업로드 또는 YouTube URL 입력).
7. [저장] 버튼을 눌러 매뉴얼 등록을 완료합니다.

**전체 사용자: 매뉴얼 열람**

1. 매뉴얼 탭으로 이동합니다.
2. 카테고리 필터를 선택하거나 검색창에 운동명을 입력합니다.
3. 목록에서 매뉴얼 카드를 선택하여 상세 화면으로 진입합니다.
4. 영상 재생 및 사진, 설명을 확인합니다.

**트레이너: 매뉴얼 수정/삭제**

1. 본인이 작성한 매뉴얼 상세 화면에서 [수정] 또는 [삭제] 버튼을 누릅니다.
2. 수정을 선택하면 기존 내용을 변경한 뒤 저장합니다.
3. 삭제를 선택하면 확인 팝업을 거쳐 삭제 처리를 완료합니다.

### 화면 설명

1. **매뉴얼 목록 화면**

   - 상단 검색바: 운동 이름으로 검색이 가능합니다.
   - 카테고리 탭: 전체, 재활, 근력, 다이어트, 스포츠 퍼포먼스로 구분됩니다.
   - 매뉴얼 카드 리스트: 썸네일, 운동명, 카테고리 태그, 작성 트레이너 이름이 표시됩니다.
   - 플로팅 버튼(FAB): 트레이너에게만 [+ 매뉴얼 추가] 버튼이 노출됩니다.

2. **매뉴얼 상세 화면**

   - 제목 영역: 운동명과 카테고리 태그가 크게 표시됩니다.
   - 정보 영역: 업로드 트레이너 이름과 등록 날짜를 보여줍니다.
   - 미디어 영역: 영상 플레이어와 가로 스크롤 형태의 사진 갤러리가 위치합니다.
   - 본문 영역: 운동법, 추천 루틴, 주의사항 텍스트가 표시됩니다.
   - 제어 영역: 작성자 본인에게만 [수정], [삭제] 버튼이 활성화됩니다.

3. **매뉴얼 작성/수정 화면**

   - 입력 폼: 카테고리 선택 드롭다운, 운동명 입력란이 있습니다.
   - 설명란: 자유로운 텍스트 입력이 가능한 멀티라인 영역입니다.
   - 미디어 첨부: 사진 추가 버튼과 업로드된 사진 목록(X 버튼으로 삭제 가능)이 제공됩니다.
   - 영상 설정: 직접 파일 업로드 버튼과 YouTube URL 입력란이 구분되어 있습니다.

4. **전체화면 뷰어**
   - 영상 플레이어: 재생, 일시정지, 탐색 바, 전체화면 토글 기능이 포함됩니다.
   - 사진 뷰어: 이미지를 크게 보여주며 좌우 스와이프로 이동이 가능합니다.

### 비즈니스 규칙

- 매뉴얼은 트레이너만 작성할 수 있으며, 모든 사용자가 열람 가능합니다.
- 수정과 삭제 권한은 해당 매뉴얼을 작성한 트레이너 본인에게만 부여됩니다.
- 운동 이름은 필수 값입니다.
- 영상은 파일 직접 업로드와 YouTube 링크 중 하나 이상을 지원합니다.
- 직접 업로드하는 영상 파일의 용량은 최대 500MB로 제한합니다.
- 사진은 한 매뉴얼당 최대 10장까지 등록할 수 있습니다.

### 엣지 케이스

- 영상 용량 초과: 업로드 시 "영상 파일은 최대 500MB입니다"라는 경고 메시지를 보여줍니다.
- 잘못된 URL: 유효하지 않은 YouTube 주소 입력 시 형식을 확인해달라는 안내를 띄웁니다.
- 삭제된 매뉴얼: 이미 배정된 '오늘의 운동' 텍스트에 매뉴얼 이름이 포함되어 있어도, 해당 텍스트 자체는 유지됩니다.

### 범위 외

- 매뉴얼에 대한 좋아요, 댓글, 공유, 북마크 등 소셜 인터랙션 기능은 제외합니다.
- 유료 매뉴얼 판매나 구독 모델은 고려하지 않습니다.
- 타인이 올린 매뉴얼을 복사하거나 템플릿으로 가져오는 기능은 포함하지 않습니다.

## 15. 운동 프로그램 배정 — 오늘의 운동

### 개요

트레이너가 회원에게 매일 수행할 운동 과제를 자유로운 텍스트 형태로 전달합니다. 복잡한 구조화 없이 간단한 메모 형식으로 소통하며, 회원은 이를 바탕으로 개인 운동을 수행합니다.

### 사용자 흐름

**트레이너: 운동 배정**

1. 일정 탭으로 이동합니다.
2. 달력에서 날짜를 선택 합니다(기본값은 오늘입니다).
3. 승인된 예약 건이 있다면 해당 예약 건을 선택합니다.
4. "오늘의 운동 배정" 화면에서 운동 루틴을 작성합니다.
5. [저장]을 누르면 회원에게 알림이 전송됩니다.

**회원: 운동 확인**

1. 홈 화면의 '오늘의 운동' 카드에서 내용을 바로 확인하거나 [전체 보기]를 누릅니다.
2. 일정 탭의 자신의 승인된 예약 건을 선택하여 확인합니다.
3. 설정 탭의 "PT 히스토리"에서 과거의 운동 내용을 조회합니다.

### 화면 설명

1. **오늘의 운동 배정 화면 (트레이너용)**

   - 대상 정보: 회원 이름과 프로필 이미지가 상단에 표시됩니다.
   - 날짜 피커: 운동을 수행할 날짜를 선택합니다.
   - 운동 루틴 입력란 : 운동 루틴을 작성합니다(운동 제목 , 세트 수, 횟수)

2. **배정 이력 화면 (트레이너용)**

   - 리스트 뷰: 날짜별로 작성된 운동 내용의 일부를 미리보기 형태로 나열합니다.
   - 관리 버튼: 각 항목 옆에 [수정]과 [삭제] 버튼이 배치됩니다.

3. **오늘의 운동 상세 화면 (회원용)**
   - 날짜 표시: 해당 운동이 배정된 날짜가 상단에 나타납니다.
   - 내용 영역: 트레이너가 작성한 전체 텍스트를 가독성 있게 보여줍니다.
   - 이동 컨트롤: 이전 날짜나 다음 날짜의 운동으로 쉽게 이동할 수 있는 화살표 버튼을 제공합니다.

### 비즈니스 규칙

- 운동 배정 및 관리 권한은 트레이너에게만 있습니다.
- 회원은 배정된 내용을 수정하거나 삭제할 수 없습니다.
- 하루에 하나의 운동 프로그램만 배정 가능하며, 재작성 시 기존 내용을 덮어씁니다.
- 배정 완료 시 해당 회원에게 즉시 인앱 알림을 보냅니다.

### 엣지 케이스

- 중복 배정: 이미 내용이 있는 날짜에 저장 시 "기존 내용을 덮어쓰시겠습니까?"라는 확인창을 띄웁니다.
- 미래 날짜 배정: 며칠 치 운동을 미리 작성하여 배정할 수 있습니다.
- 연결 해제: 트레이너와 회원의 연결이 끊겨도 기존에 배정된 이력은 데이터베이스에 보관됩니다.

### 범위 외

- 회원이 운동 완료 버튼을 누르거나 수행 결과를 기록하는 기능은 제외합니다.
- 매뉴얼 화면에서 특정 운동을 끌어다 놓아 배정하는 드래그 앤 드롭 기능은 포함하지 않습니다.

## 16. 회원 운동 메모

### 개요

트레이너가 회원의 특이사항, 컨디션, 운동 시 주의점 등을 기록하는 전용 공간입니다. 트레이너의 관리 효율을 높이고 회원에게는 본인의 상태에 대한 피드백을 전달하는 용도로 활용됩니다.

### 사용자 흐름

**트레이너: 메모 작성 및 관리**

1. 회원 상세 정보 화면에서 '메모' 섹션을 확인합니다.
2. 우측 하단의 FAB.[메모 추가] 버튼을 눌러 날짜와 메모 내용을 입력한 뒤 저장합니다.
3. 목록에서 특정 메모를 탭하여 상세 내용을 확인하거나 수정, 삭제합니다.

### 화면 설명

1. **메모 목록 화면 (트레이너용)**

   - 회원 식별: 상단에 회원 이름이 명시됩니다.
   - 카드 리스트: 작성 날짜와 메모 내용의 앞부분(약 2줄)을 보여줍니다.
   - 추가 버튼: 우측 하단에 플로팅 형태로 [메모 추가] 버튼이 위치합니다.

2. **메모 작성/수정 창**

   - 날짜 선택: 메모와 연결된 날짜 및 시간을 지정합니다.
   - 텍스트 필드: 제약 없는 자유로운 작성이 가능한 입력 창입니다.

### 비즈니스 규칙

- 메모의 모든 편집 권한(생성, 수정, 삭제)은 트레이너에게 있습니다.
- 한 날짜에 여러 개의 메모를 남길 수 있습니다.
- 메모는 텍스트 위주로 구성됩니다.

### 엣지 케이스

- 메모 삭제: 실수 방지를 위해 삭제 전 반드시 확인 절차를 거칩니다.

### 범위 외

- 회원이 트레이너에게 메모를 남기는 기능은 포함하지 않습니다.
- 무게 변화나 체지방률 등 수치 데이터의 시계열 그래프 표시는 제외합니다.
- 특정 키워드로 메모를 검색하는 기능은 구현하지 않습니다.

### 용어 정의

- **PT (Personal Training)**: 트레이너와 회원이 정해진 시간에 만나 1:1로 진행하는 운동 세션입니다.
- **트레이너**: 서비스를 통해 회원을 관리하고 운동 매뉴얼을 제공하는 전문가 사용자입니다.
- **회원**: 트레이너에게 운동 지도를 받는 사용자이며, 한 번에 한 명의 트레이너와만 연결됩니다.
- **승인/거절**: 회원의 예약 요청에 대해 트레이너가 수락하거나 거부하여 일정을 확정하는 단계입니다.
- **매뉴얼**: 운동 자세와 팁을 담은 트레이너 제작 콘텐츠로, 모든 사용자가 학습용으로 활용합니다.
- **오늘의 운동**: 당일 수행해야 할 운동 목록을 담은 간단한 텍스트 공지입니다.
- **초대 코드**: 트레이너가 특정 회원을 자신과 연결하기 위해 발급하는 고유 식별 번호입니다.
