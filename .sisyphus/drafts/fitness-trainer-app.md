# Draft: 헬스 트레이너 앱 기능문서

## Requirements (confirmed)
- **사용자 유형**: 트레이너 + 회원 (2종류만, 관리자 없음)
- **플랫폼**: Vue.js 웹 + Flutter 웹앱 (Flutter에서 Vue.js 웹을 WebView로 호출)
- **인증**: 카카오 OAuth (회원가입/로그인)
- **캘린더**: 일정 관리용 (알림 기능 X)
- **채팅**: 1:1 채팅 + 파일/이미지 전송 (그룹채팅 X)
- **PT 예약**: 회원 예약 요청 → 트레이너 승인 → 등록 확정
- **운동 매뉴얼**: 트레이너가 직접 업로드, 카테고리: 재활/근력/다이어트/스포츠 퍼포먼스
- **매뉴얼 콘텐츠**: 영상 + 사진 + 제목 + 설명 (자세, 세트/횟수, 주의사항)
- **영상 업로드**: 직접 업로드 + YouTube 링크 모두 지원
- **트레이너-회원 관계**: 1:N (한 트레이너가 여러 회원 관리, 회원은 한 트레이너에게만 배정)
- **트레이너 매칭**: 초대 링크/코드 + 앱 내 검색 모두 가능
- **예약 시간 단위**: 트레이너가 직접 설정 (30분/1시간 등 커스텀)
- **근무 시간**: 트레이너가 요일/시간별 가능 시간 미리 설정
- **예약 취소**: 자유 취소 (예약 시작 전까지 언제든)
- **매뉴얼 공개 범위**: 전체 공개 (모든 사용자 열람)
- **결제**: 간단한 수납 기록만 (실제 결제 X)
- **회원 운동 기록**: 트레이너가 회원별 간단한 메모/노트 작성
- **트레이너 프로필**: 기본 정보 (이름, 프로필 사진, 전문 분야)
- **회원 프로필**: 기본 + 신체정보(key/몸무게/나이) + 운동 목표/특이사항
- **운동 프로그램 배정**: 트레이너가 회원별 간단한 루틴 전달 ('오늘의 운동' 등)
- **PT 횟수 관리**: 트레이너가 회원별 등록 횟수/잔여 횟수 관리

## Technical Decisions
- Vue.js 웹 (프론트엔드 메인)
- Flutter는 웹뷰로 Vue.js 웹을 감싸는 형태
- 백엔드: .NET 9 ASP.NET Core (AGENTS.md 기준)

## All Questions Resolved
- 트레이너-회원 관계: **1:N**
- 근무 가능 시간 설정: **예, 미리 설정**
- 예약 단위: **트레이너가 설정**
- 트레이너 매칭: **초대 링크 + 검색 모두**
- 예약 취소: **자유 취소**
- 매뉴얼 공개: **전체 공개**
- 결제: **수납 기록만**
- 영상: **직접 업로드 + YouTube 링크**
- 운동 기록: **간단한 메모**
- 트레이너 프로필: **기본 정보**
- 회원 프로필: **기본 + 신체 + 목표**
- 운동 프로그램: **간단한 배정**
- PT 횟수: **횟수 관리 필요**
- 매뉴얼 상세: **영상/사진 + 제목 + 설명**
- 문서 언어: **한국어 (기능 정의서)**
- 인앱 알림: **필요 (예약요청/채팅/루틴배정 등에 대한 앱 내 알림)**
- 온보딩: **카카오 로그인 → 역할 선택(트레이너/회원) → 프로필 입력 → 완료**
- 운동 프로그램 형태: **자유 텍스트 메모 (구조화 X)**

## Scope Boundaries
- INCLUDE: 회원가입/로그인(카카오 OAuth), 캘린더, 1:1 채팅(파일/이미지 전송), PT예약(요청/승인/취소), 운동 매뉴얼(영상+사진), 트레이너 프로필, 회원 프로필, 수납 기록, 운동 메모, 운동 프로그램 배정, PT 횟수 관리, 트레이너 검색/초대
- EXCLUDE: 푸시 알림, 온라인 결제(PG사), 그룹 채팅, 후기/평점 시스템, 상세 운동 추적(세트/횟수/중량), 상세 트레이너 프로필(자격증/경력/포트폴리오)
